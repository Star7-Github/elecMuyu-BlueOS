# BlueOS上的电子木鱼应用
## 使用说明
### 首页
 - **标题`电子木鱼`**
   - 点击进入`设置`页
 - **木鱼**
   - 点击敲击，功德+1
 - **功德值**
   - 点击可重置计数
### 设置页
 - 可重置计数、开关振动、开关音效
 - 可查看版本号、更新日志、许可信息、版权声明
   - 点击`版本号`可获取更新
 - 可查看开发者信息
   - 点击`联系方式`可扫码发邮件
